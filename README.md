Web Scraping - Code Alpha Internship Task 1

Description

This project demonstrates basic web scraping using Python and BeautifulSoup. The script extracts quotes and author names from a public website and stores the data in a CSV file.


Tools Used

Python

BeautifulSoup

Requests

Visual Studio Code

Website Scraped

http://quotes.toscrape.com/

Features
Extracts quotes automatically
Extracts author names
Saves data into CSV format
Simple and beginner-friendly code
How to Run

Install required libraries:

    pip install requests beautifulsoup4

Run the script:

    python scrape.py
